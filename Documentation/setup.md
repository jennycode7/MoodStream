# Setup Guide

---

## Installation

```bash
git clone https://github.com/jennycode7/MoodStream
cd MoodStream
pip install -r requirements.txt
```

---

## Environment Variables

Create `.env` file:

```
DEBUG=True
SECRET_KEY=your_secret
DB_NAME=your_db
DB_USER=your_user
DB_PASSWORD=your_password
```

---

## Run Server

```bash
python manage.py runserver
```
